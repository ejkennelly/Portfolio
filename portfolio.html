This app is designed to geolocate the user, provide them with options of what type of location they're looking to go to, and provide them with options and a map showing them where that destination is. It was made in conjunction with Satya Kavya and Ted Peters.

"Random Password Generator"

This project creates a randomly generated password between 8 and 128 characters based on the users desired parameters. Alerts pop up when the generate password is clicked that ask a user what type of characters they would like. When all conditions are met, a password appears.

Day Planner

This web day planner shows you a 9am-5pm day, with each time block displaying a different color based on if it's in the future, past, or present. When a user enters their task in time block and clicks save, that input is saved to local storage to refer to later.
